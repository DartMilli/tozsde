🚀 QUICK START: Deploy to Raspberry Pi
════════════════════════════════════════════════════════════════════════════════

PHASE 1: Physical Setup (15 min)
  1. Flash Raspberry Pi OS Lite 64-bit to SD card
  2. Boot, enable SSH, update system
  3. Note Pi's IP address

PHASE 2: Application Deployment (45 min)
  1. SSH to Pi: ssh pi@raspberrypi.local
  2. Clone repo: git clone <repo> ~/tozsde_webapp
  3. Run deploy: bash deploy_rpi.sh
     ✓ Installs Python 3.11
     ✓ Creates venv & installs dependencies
     ✓ Sets up Flask API (systemd service)
     ✓ Configures cron jobs (daily, weekly, monthly)
     ✓ Enables health checks & monitoring
     ✓ Starts all services

PHASE 3: Verification (5 min)
  curl http://raspberrypi.local:5000/api/health
  # Expected: {"status": "healthy", ...}
