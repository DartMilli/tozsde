### 2.2 Health Check Bash Script
**File:** `scripts/health_check.sh` (NEW)  
**Dependencies:** health_check.py, curl, Python venv  
**Time:** 0.5 hours

**Requirements:**
- [x] Called by cron every 5 minutes
- [x] Checks Flask API: `curl http://localhost:5000/api/health`
- [x] Checks database accessibility
- [x] Logs results to `logs/health_check.jsonl`
- [x] Restarts service if API is down (with 5-min delay before restart)
- [ ] Alerts via email if critical issues (optional/future)

**Bash code:**
```bash
#!/bin/bash
# Health check every 5 minutes (from crontab)

VENV=/home/pi/tozsde_webapp/venv
API_URL="http://localhost:5000/api/health"
LOG_FILE="/home/pi/tozsde_webapp/logs/health_check.jsonl"

# Activate venv
source $VENV/bin/activate

# Check API
HTTP_STATUS=$(curl -s -o /dev/null -w "%{http_code}" $API_URL)

if [ "$HTTP_STATUS" != "200" ]; then
    echo "{\"timestamp\": \"$(date -Iseconds)\", \"status\": \"down\", \"http_code\": $HTTP_STATUS}" >> $LOG_FILE
    # Restart service (with delay)
    systemctl restart tozsde-api.service
else
    echo "{\"timestamp\": \"$(date -Iseconds)\", \"status\": \"ok\", \"http_code\": 200}" >> $LOG_FILE
fi
```
