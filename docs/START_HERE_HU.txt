🚀 GYORS KEZDÉS: Telepítés Raspberry Pi-re
════════════════════════════════════════════════════════════════════════════════

FÁZIS 1: Fizikai Setup (15 perc)
  1. Raspberry Pi OS Lite 64-bit formázása SD-kártyára
  2. Boot, SSH engedélyezése, rendszer frissítése
  3. Pi IP-cím feljegyzése

FÁZIS 2: Alkalmazás Telepítés (45 perc)
  1. SSH-zd be: ssh pi@raspberrypi.local
  2. Klónozd a repot: git clone <repo> ~/tozsde_webapp
  3. Futtasd a telepítést: bash deploy_rpi.sh
     ✓ Python 3.11 telepítése
     ✓ Venv létrehozása & függőségek telepítése
     ✓ Flask API setup (systemd service)
     ✓ Cron feladatok konfigurálása (napi, heti, havi)
     ✓ Health check-ek engedélyezése & monitoring
     ✓ Összes service indítása

FÁZIS 3: Ellenőrzés (5 perc)
  curl http://raspberrypi.local:5000/api/health
  # Várható: {"status": "healthy", ...}

K: Hol a teljes Raspberry Pi setup útmutató?
V: docs/01_deployment/RASPBERRY_PI_SETUP_GUIDE_HU.md

K: Hogyan telepítsek a Raspberry Pi-re?
V: Futtasd: bash deploy_rpi.sh (10 perc, teljes automatizálás)

K: A tesztek nem működnek?
V: Ellenőrizd: pytest.ini és tests/conftest.py a fixtúrákhoz

K: Az API nem válaszol?
V: SSH-zd be a Pi-be, ellenőrizd: sudo journalctl -u tozsde-api.service -n 50

K: Cron feladat nem futott?
V: Ellenőrizd a naplókat: cat ~/tozsde_webapp/logs/cron_daily.log

K: Lemez megtelt figyelmeztetés?
V: Töröld a régi naplókat: rm ~/tozsde_webapp/logs/*.gz

K: SSH-val nem tudsz csatlakozni?
V: Ellenőrizd az IP-t: hostname -I (Pi-n), ping a Pi-t, SSH engedélyezve legyen
